# ciya-dev
Claude Code in your area

## Scripts

Run from the `main` worktree:

| Script | Usage | Description |
|--------|-------|-------------|
| `scripts/hi.sh` | `scripts/hi.sh <branch>` | Create worktree, cd into it, start Claude Code |
| `scripts/bb.sh` | `scripts/bb.sh <branch-or-path>` | Remove worktree, delete branch, prune remotes |

## Slash Commands

Type inside a Claude Code session:

| Command | Description |
|---------|-------------|
| `/go`   | Start a new task or resume an in-progress task |
| `/fb`   | Address PR review feedback |
| `/ty`   | Approve and proceed to the next workflow step |
